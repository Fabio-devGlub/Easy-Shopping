# 🌐 Landing Page Web

Projeto de **Landing Page** desenvolvido com **HTML e CSS**, com foco em praticar estruturação de páginas web e estilização moderna.

A página apresenta um layout simples e organizado, utilizando imagens e estilos para criar uma interface visual agradável.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

---

## ⚙️ Funcionalidades

* 🎨 Layout estilizado com CSS
* 🖼️ Uso de imagens para composição visual
* 📱 Estrutura simples e fácil de navegar
* ⚡ Página leve e rápida

---

## 📂 Estrutura do projeto

```
landing-page
│
├── index.html
├── stayles.css
│
└── img
    └── (imagens utilizadas no projeto)
```

---

## ▶️ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/seu-usuario/landing-page.git
```

2. Entre na pasta do projeto

```
cd landing-page
```

3. Abra o arquivo

```
index.html
```

no navegador.
